══════════════════════════════════════════════════════════════════════════════════════
                         各阶段 "0" 行为 + 窗口画面
══════════════════════════════════════════════════════════════════════════════════════

  阶段                    │ USB用过? │ 窗口画面        │ 收到 "0" 的行为
  ───────────────────────┼─────────┼────────────────┼──────────────────────────
  USB 预热 (等指令)        │  ✗       │ USB原始画面      │ 无视
  USB 检测中              │  ✗       │ USB检测画面      │ 无视
  USB 完成 → 等 "2m"      │  ✓       │ USB检测画面(idle)│ —
  D435 预热 (等 1m/2m)    │  ✓       │ D435原始画面     │ 保持推流+预览
  D435 检测中 (2m/1m)     │  ✓       │ D435检测画面     │ 暂停检测，20s计时
  D435 暂停中             │  ✓       │ D435原始画面     │ timer≤20s: 等1m/2m恢复
                           │         │                │ timer>20s: 释放D435
  等待 data.txt 空        │  —       │ 无窗口(无相机)   │ —
  新一轮 USB 预热          │  ✗       │ USB原始画面      │ 无视


══════════════════════════════════════════════════════════════════════════════════════
                       关 键 状 态 变 量
══════════════════════════════════════════════════════════════════════════════════════

  变量               │ 位置         │ 作用
  ──────────────────┼──────────────┼────────────────────────────────
  _usb_used         │ ModeState    │ USB聚类完成→True; 区分0无视/暂停
  _pause_start      │ ModeState    │ D435暂停计时起点
  _d435_released    │ Orchestrator │ D435已释放, 等data.txt清空
  _d435_camera      │ Orchestrator │ D435相机引用, 预热时非None
  start_event       │ main()       │ 管线预热→检测的触发信号
  stop_event        │ Orchestrator │ 管线退出信号 (每轮新建)
